**1. 프로젝트 개요**

**--------------------------------------------------**

**--------------------------------------------------**

**3. 파일 역할**

**--------------------------------------------------**

**index.html → 화면 구조**

**style.css → 디자인**

**app.js → 기능 (API 호출, 로그인)**

**images → 이미지 저장**

**관리문서 → 구조 설명**



**--------------------------------------------------**

**4. 화면 요소 (id 기준)**

**--------------------------------------------------**

**eyebrowText → 상단 작은 문구**

**siteTitle → 메인 제목**

**noticeText → 설명 문구**



**featureText1 → 기능 설명1**

**featureText2 → 기능 설명2**

**featureText3 → 기능 설명3**



**loginTitle → 로그인 제목**

**loginDesc → 로그인 설명**



**mainName → 본캐명 입력칸**

**loginButton → 로그인 버튼**

**loginResult → 로그인 결과**



**adminCode → 관리자 코드 입력**

**adminButton → 관리자 버튼**

**adminResult → 관리자 결과**



**footerNote → 하단 안내문**



**logoImagePc → PC 로고**

**logoImageMobile → 모바일 로고**

**leftPanel → 좌측 배경**

**--------------------------------------------------**

**7. JS 주요 기능**

**--------------------------------------------------**

**callApi → Apps Script 호출**



**initPage → TEXT / IMAGE 불러오기**

**login → 본캐명 로그인**

**adminLogin → 관리자 로그인**



**setText → 화면 글자 변경**

**setPlaceholder → input 안내문 변경**

**setMessage → 결과 메시지 출력**



**--------------------------------------------------**

**8. Apps Script action**

**--------------------------------------------------**

**init → 초기 데이터 불러오기**

**login → 로그인 처리**

**adminLogin → 관리자 인증**



**--------------------------------------------------**

**9. 수정 방법 (핵심 흐름)**

**--------------------------------------------------**



**\[글자 바꾸기]**

**1. index.html에서 id 찾기**

**2. app.js에서 key 확인**

**3. 구글시트 TEXT 탭 수정**



**예:**

**loginTitle → login\_title 수정**



**\[이미지 바꾸기]**

**1. app.js에서 key 확인**

**2. IMAGE 탭 url 수정**



**예:**

**IMG\_LOGO 변경**



**--------------------------------------------------**

**10. 자주 터지는 문제**

**--------------------------------------------------**

**1. css 경로 틀림 → 디자인 깨짐**

**2. js 안 불림 → 버튼 안됨**

**3. 이미지 경로 틀림 → 안보임**

**4. TEXT key 오타 → 문구 안나옴**



**--------------------------------------------------**

**11. 운영 규칙 (중요)**
